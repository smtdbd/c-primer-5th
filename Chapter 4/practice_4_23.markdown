### 因为运算符的优先级问题，下面的这条表达式无法通过编译。根据4.12节中的表(第147页)指出它的问题在哪里？应该如何修改？
    string s = "word";
    string p1 = s + s[s.size() - 1] == 's' ? "" : "s";
***
本程序试图判断s的最后一个字符是否为's'，若不是，则加上一个's'  
但是由于+运算符的优先级大于==的优先级，使得其运算顺序为  

    string p1 = (s + s[s.size() - 1]) == 's' ? "" : "s";
就发生了试图将一个string和一个char比较的错误，应该改为

    string p1 = s + (s[s.size() - 1] == 's' ? "" : "s");
